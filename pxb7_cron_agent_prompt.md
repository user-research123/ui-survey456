# 螃蟹账号平台（pxb7.com）日报自动化任务指令

**目标**：抓取 pxb7.com “王者荣耀世界”最新发布的 100 个商品数据，分析并更新本地及 GitHub 上的竞品报告。

## 执行步骤

### 1. 数据爬取
1.  **打开页面**：使用 `use_browser` 访问 `https://www.pxb7.com/game/wzsj` (假设路径，若无效则搜索“王者荣耀世界”)。
2.  **注入 Cookie**：若页面未加载数据，尝试从本地存储或环境变量读取有效 Cookie 并注入。若失败，记录错误并通知用户。
3.  **排序**：点击“最新发布”按钮。
4.  **滚动加载**：使用 JS `evaluate` 滚动页面，每次滚动后等待 2s，重复直到提取到至少 100 个商品元素。
5.  **提取数据**：提取每个商品的标题、价格、平台（QQ/微信）、链接。保存为 `pxb7_products_YYYYMMDD.json`。

### 2. 数据分析
1.  **运行分析脚本**：调用 Python 脚本 `analyze_pxb7_data.py` 处理 JSON 文件。
2.  **生成总结**：计算价格中位数、区间分布百分比、平台占比、ID 命名特征（单字/双字比例）。

### 3. 报告更新
1.  **定位文件**：找到 `wangzhe_report/index_with_tabs.html`。
2.  **添加日期区块**：
    *   在顶部日期切换栏添加当天日期按钮（如“4月9日”）。
    *   在内容区添加新的 `div` 区块，ID 为 `competitor-YYYYMMDD`。
    *   区块内包含三个子卡片：竞品一：螃蟹、竞品二：盼之、竞品三：闲鱼。
3.  **填入数据**：将步骤 2 生成的螃蟹分析总结填入“竞品一：螃蟹”卡片中。
4.  **激活新日期**：修改 JS 逻辑，使页面加载时默认显示当天日期。

### 4. 生成与推送
1.  **生成 JSON**：确保 `pxb7_analysis_YYYYMMDD.json` 已生成。
2.  **Git 推送**：
    *   `git add .`
    *   `git commit -m "Update pxb7 daily report for YYYY-MM-DD"`
    *   `git push origin main` (使用缓存的凭证)。
3.  **通知**：推送成功后，通过钉钉通知用户“衡初”任务完成。

## 注意事项
*   **反爬处理**：pxb7.com 有强反爬，若浏览器自动化失败，请检查 Cookie 有效性。
*   **数据完整性**：若某项数据缺失，请在报告中标记“数据获取失败”，不要留空或复制旧数据。
*   **手动触发**：此任务由用户手动触发，执行前请确认当前环境已登录或有有效凭证。
