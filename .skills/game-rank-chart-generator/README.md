# Game Rank Chart Generator - 游戏榜单曲线图生成器

## 概述

这是一个专门用于生成游戏榜单排名趋势曲线图的skill,基于《王者荣耀世界》项目实践沉淀而成。它将七麦数据等来源的榜单CSV/JSON数据转换为符合规范的离线可用SVG曲线图HTML文件。

## Skill ID

`b82b8a13-9d46-49a5-8a84-5c157e23749d`

## 核心特性

✅ **Y轴排名正序**: 0在最上方,数值越大越靠下(符合排名直觉)  
✅ **X轴简化格式**: 仅显示月日(如03-28),避免日期堆叠  
✅ **紧凑设计**: 图表高度350px,适合嵌入报告  
✅ **悬停交互**: tooltip显示完整时间和排名信息  
✅ **完全离线**: 纯SVG实现,无需ECharts/D3.js等外部库  
✅ **响应式**: 自动适配窗口resize事件  
✅ **数据来源标注**: 底部固定显示"数据来源:七麦数据"  

## 使用方式

### 触发关键词

- "生成榜单曲线图"
- "制作排名趋势图"
- "游戏榜单可视化"
- "七麦数据图表"
- "免费榜曲线"
- "畅销榜曲线"

### 输入数据格式

**CSV格式:**
```csv
time,game_rank
2026-03-27 00:00,
2026-03-28 05:00,95
2026-03-30 13:00,3
```

**JSON格式:**
```json
[
  {"time": "2026-03-27 00点00分", "game": ""},
  {"time": "2026-03-28 05点00分", "game": "95"}
]
```

### 输出

独立的HTML文件,包含:
- 完整的CSS样式(内联)
- SVG曲线图结构
- JavaScript绘制逻辑(rawData + createChart函数)
- 可直接在浏览器打开或嵌入到其他HTML报告中

## 技术实现

- **纯原生JavaScript**: 无外部依赖
- **SVG动态生成**: 使用createElementNS创建SVG元素
- **数据预处理**: 过滤空值、转换时间格式、提取月日
- **坐标计算**: Y轴向上取整到10的倍数,X轴标签间隔动态计算
- **事件绑定**: mouseenter/mouseleave/mousemove控制tooltip

## 应用场景

1. **日报自动化**: 每日从七麦数据爬取榜单后自动生成曲线图
2. **报告更新**: 将新曲线图嵌入到GitHub Pages报告中
3. **趋势分析**: 直观展示游戏排名变化趋势
4. **竞品对比**: 可同时生成多个游戏的曲线图进行对比
