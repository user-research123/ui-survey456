怎么关闭自动付费
苹果设备：在 iCloud 支付功能中关闭。路径为：设置 → iTunes Store 与 App Store → Apple ID → 查看 Apple ID → 账户设置 → 订阅。
安卓设备：在GooglePlay订阅功能中关闭。路径为：GooglePlay侧边栏→ 订阅。
因操作系统升级等原因导致路径调整的而您无法正确关闭自动续费的，您可以咨询设备厂家或系统服务商，也可以私信联系我们的 客服，我们将尽力为您提供解决方案。
关闭本功能视为您撤销给予穿梭Transocks的从第三方支付账户扣除费用的授权。但此撤销不影响穿梭Transocks在您撤销前已完成的扣款。
无论因何原因，您终止使用相应付费产品服务的，相应已开通的自动续费功能都需要您自行操作关闭。如您未关闭，穿梭Transocks将正常进行自动扣款的操作，一旦扣款成功，穿梭Transocks将为您正常延续相应服务。
