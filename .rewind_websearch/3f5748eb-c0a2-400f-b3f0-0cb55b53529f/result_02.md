计费概述  #
计费项  #
WebSocket 数据传输费 ：智能全球加速针对 WebSocket 服务收取 WebSocket 数据传输费。计费支持 按流量计费 、 按峰值带宽计费 。系统默认以按流量计费方式对 WebSocket 数据传输进行计费。如需开通 WebSocket 按峰值带宽计费，请联系您的客户经理或 提交工单 。 不计请求数费用 ：与标准的 HTTP/HTTPS 请求不同，WebSocket 的访问流量不产生请求数费用。
注意
WebSocket 计费为增值服务费用，将与 WebSocket 访问产生的普通公网流量费叠加计算。
计费区域  #
WebSocket 数据传输费用的单价因 WebSocket 流量接入的加速区域而不同，即为用户访问流量进入智能全球加速边缘节点的区域。以下为各计费区域的具体详细说明：

按流量计费  #
计费描述  #
计费方式 ：WebSocket 流量计费的付费模式是按量计费（后付费）模式。该计费模式以用户实际产生的WebSocket 流量为计费用量。 计费周期 ：WebSocket 流量计费的计费周期为 1 小时。系统会统计每个小时内您的流量使用情况，在计费周期结束后的两小时左右结算并实施扣费。 阶梯定价 ：系统会对一个自然月内使用的流量持续累加，累计流量所落入的阶梯区间决定了这部分流量的单价。累计流量每月 1 日 00:00:00 清零。
价格详情  #
在一个计费周期内，流量费用 = 流量 * 区间价格。区间价格取决于当月累计流量所在的定价区间。具体价格如下：

计费示例  #
示例 1：在 2021-03-01 00:00:00 至 2021-03-01 00:59:59 的计费周期内，来自中国内地的 WebSocket 访问产生的流量为 60GB。截至到 2021-03-01 00:59:59，当月累积流量为 60GB。该 60GB 流量处于 0 - 10 TB 的定价区间。该计费周期的费用为：60 * 2.1 = 126 元。
示例 2：在 2021-03-11 00:00:00 至 2021-03-11 00:59:59 的计费周期内，来自中国内地的 WebSocket 访问产生的流量为 90GB。截至到 2021-03-11 00:59:59，当月累积流量为 10050GB。该 90GB 的流量中，前面的 40GB 处于 0 - 10TB 的定价阶梯，后面的 50GB 处于 10TB - 50TB 的定价阶梯。该计费周期的费用为：40 * 2.1 + 50 * 2 = 184 元。
按峰值带宽计费  #
计费描述  #
计费方式 ：WebSocket 峰值带宽计费的付费模式是按量计费（后付费）模式。以计费日当天的峰值带宽作为该计费日的计费用量。
计费周期 ：WebSocket 带宽峰值计费的计费周期为 1 天。计费日的 00:00:00～23:59:59 产生的使用量，在次日的 18:00:00 左右结算并实施扣费。
阶梯定价 ：计费日当日的峰值带宽所在的阶梯区间决定了当日峰值带宽用量的单价。
价格详情  #
系统将计费日当天 00:00:00 ~ 23:59:59 分割成每个长度 5 分钟的共 288 个计费样本，并将这些计费样本中带宽的最大值作为当天的带宽峰值进行计费。
 在一个计费周期内，峰值带宽费用 = 峰值带宽 * 区间价格。区间价格取决于当日峰值带宽用量所在的定价区间。具体价格如下：
