# <em>谷子</em>经济爆火：艾媒咨询数据解读，<em>二次元</em>年货引爆千亿新赛道-网易

URL: https://m.163.com/dy/article/KNJEH9OL0514A1HE.html

Query: 谷子市场 二次元周边市场规模 2025 2026 行业报告

---

2026 春节消费新风向已清晰显现，新华网新春趋势数据显示，二次元 “谷子” 跻身最具创意年货 TOP3，徽章、立牌、色纸等周边成为 Z 世代拜年、送礼、收藏的热门选择。从线上拼团到线下谷店密集开业，从圈层爱好到全民流行，“谷子经济” 已突破小众边界，成长为规模超 2000 亿的消费新风口，更入选《咬文嚼字》2025 年度十大流行语，成为年度现象级消费热点。
眼下，上海、成都、合肥等多地积极布局二次元消费场景，以 “谷子经济” 激活文旅消费新活力。伴随《黑神话：悟空》《哪吒之魔童闹海》等国产 IP 火爆出圈，国产谷子备受年轻群体追捧，市场迎来快速扩容期，一场由二次元文化驱动的消费变革正在席卷市场。
一、市场规模高速增长，谷子经济迈入千亿时代
iiMedia Research（艾媒咨询）数据显示，中国谷子经济呈现稳健增长态势，2025 年市场规模达到 2021 亿元，同比增长 19.7%，正式迈入两千亿量级。艾媒咨询分析师指出，谷子经济持续扩容，既是中国二次元产业走向成熟的重要标志，也是情绪消费崛起的典型缩影。
谷子以 IP 为情感纽带，将虚拟热爱转化为实体消费，通过收藏、交换、社群互动等场景，精准满足 Z 世代情感寄托、自我表达与圈层认同需求，推动情绪经济向细分领域深度渗透。
依托成熟的 IP 孵化体系与全链条运营能力，中国 IP 授权市场稳步扩张。数据显示，2024 年中国 IP 授权市场规模达 1686 亿元；潮流玩具行业实现跨越式增长，市场规模从 2015 年 63 亿元增至 2024 年 763 亿元，从小众收藏升级为大众消费，为谷子经济提供坚实产业基础。
二、用户基数庞大，泛二次元产业持续扩容
艾媒咨询数据显示，中国泛二次元用户规模稳步增长，2025 年达 5.26 亿人，同比增长 2.65%，预计 2029 年将增至 5.70 亿人。庞大的用户基础，为谷子经济提供稳定需求支撑。
与此同时，泛二次元及周边市场增长势头强劲。2019 年市场规模 2983 亿元，2025 年升至 6521 亿元，预计 2029 年将达到 8344 亿元。分析师认为，年轻群体精神消费需求持续提升，泛二次元产业已成为年轻人核心消费趋势，长期增长空间广阔。
三、国产 IP 崛起，谷子经济走向主流与精品化
从消费逻辑看，谷子经济的核心驱动力已从单纯商品购买，转向情感价值与 IP 认同。Z 世代与童心成年人成为消费主力，精神需求驱动高频复购，形成 “高粘性、高复购、高溢价” 的消费特征。
在内容供给端，国产 IP 强势崛起，打破海外 IP 主导格局，国漫、游戏、影视 IP 衍生谷子占据市场主流位置。线下门店、主题商圈、快闪店不断落地，谷子消费场景持续拓宽，从线上走向线下、从圈层走向大众，成为城市潮流消费的重要组成部分。
如今，谷子经济已从亚文化圈层走向大众消费舞台，成为驱动新消费、新文化、新文旅的重要力量。在用户扩容、IP 升级、场景丰富的多重驱动下，谷子经济将保持稳健增长，进一步推动文创产业与实体经济融合，成为中国消费市场中极具活力与潜力的黄金赛道。
